Few-shot 示例（v4 澄清）：

示例1
输入：整理一下最近关于门店巡检机器人的讨论，做个输出。
输出：
{
  "questions": [
    "你希望整理为文档、PPT，还是两者都要？",
    "希望覆盖最近一周、一个月，还是具体时间段？"
  ]
}

示例2
输入：把昨天会上的内容整理成PPT。
输出：
{
  "questions": [
    "这份PPT主要面向谁汇报（老板、客户还是项目组）？",
    "你希望控制在多少页左右？"
  ]
}

示例3
输入：帮我写一篇 Spring AI Alibaba 的技术介绍，给后端开发看。
输出：
{
  "questions": [
    "你希望重点介绍框架能力、接入方式，还是结合当前项目的落地架构？"
  ]
}

示例4
输入：基于已选消息整理本周项目风险，输出给项目组看的文档。
输出：
{
  "questions": []
}
